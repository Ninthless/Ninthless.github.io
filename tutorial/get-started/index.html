入门

欢迎来到 Vibe Coding 教程。本章帮你在最短时间内搭好环境、理解核心概念，并完成第一个真实任务。

本章内容

章节 内容
环境准备 安装必备工具，配置本地开发环境
核心概念 理解 Vibe Coding 的关键术语和思维模型
快速上手 完成第一个端到端的 Vibe Coding 任务

前置要求

  • 有基本的编程经验（任意语言）
  • 能使用命令行终端
  • 有 GitHub 账号（用于版本控制和发布）
